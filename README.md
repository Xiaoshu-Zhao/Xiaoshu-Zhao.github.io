# Xiaoshu Zhao - Personal Website

A minimalist, tech-inspired personal portfolio website showcasing my work as a Software Engineer.

## Design Philosophy

This website follows a **Minimalist + Tech Aesthetic** design approach:

- **Color Palette**: Deep gray (#1a1a1a), pure white (#ffffff), and tech blue (#0066ff)
- **Typography**: IBM Plex Mono for headers (technical feel), Inter for body text (readability)
- **Layout**: Asymmetric design with large whitespace and geometric elements
- **Interactions**: Smooth transitions and subtle hover effects

## Features

- **Hero Section**: Introduction and quick stats
- **Projects Showcase**: Featured projects with descriptions and links
- **Skills & Technologies**: Organized by category (Languages, Frontend, Backend, Tools)
- **Contact Section**: Social media links and contact information
- **Responsive Design**: Fully responsive across all devices

## Technologies Used

- **Frontend**: React 19, Tailwind CSS 4
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **Fonts**: IBM Plex Mono, Inter (via Google Fonts)

## Built With

- Vite (build tool)
- TypeScript
- React Router (Wouter)
